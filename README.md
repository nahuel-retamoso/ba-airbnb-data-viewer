## Visualizador de publicaciones de Airbnb

Framework y librerias usadas:

  - React
  - ChakraUI
  - Apollo
  - Redux
  - Leaflet

##### Permite seleccionar los datos que se van a pedir al backend con graphQL. El dataset esta almacenado con mongoDB.
